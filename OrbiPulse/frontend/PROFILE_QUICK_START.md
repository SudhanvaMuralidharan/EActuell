# Quick Start - Profile Feature

## 🎯 Accessing Your Profile

1. **Open the app** - Go to the main dashboard
2. **Find the profile icon** - Top-right corner (👤 icon)
3. **Tap to open** - Profile modal slides up

## 📱 Profile Menu Options

### ⚙️ Settings
Opens the Settings screen where you can:
- Update profile information
- Manage notifications
- Change system preferences
- Access support resources

### 🎨 Theme
Toggle dark mode directly in the modal:
- Tap the switch to enable/disable
- Changes apply immediately
- Persists across app restarts

### ℹ️ About
View app information:
- Version number
- Feature list
- Developer details
- Company website

## 🔄 Quick Actions

| Action | How To |
|--------|--------|
| Open Profile | Tap 👤 icon |
| Close Modal | Tap X or outside |
| Go to Settings | Tap Settings option |
| Toggle Dark Mode | Use switch in modal |
| View About | Tap About option |
| Go Back | Use back arrow in screens |

## 💡 Tips

- Profile icon is always visible in top-right
- Modal closes automatically when navigating
- Dark mode preference is saved
- All screens have back navigation
