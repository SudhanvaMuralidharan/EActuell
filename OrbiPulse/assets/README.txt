This folder should contain:
- icon.png        (1024×1024 app icon)
- splash.png      (recommended for splash screen)
- adaptive-icon.png (Android adaptive icon foreground)

For the hackathon, you can use any placeholder PNG or generate
an icon from the OrbiPulse brand colours (#050D1A background, #00E5A0 accent).

Quick placeholder command (requires ImageMagick):
  convert -size 1024x1024 xc:#050D1A icon.png
